CRISPR Web Tool

How to Use This Website
  1. Enter your gene IDs in the text box below. You can enter multiple gene IDs separated by commas (e.g., GENE1, GENE2, GENE3).
  2. Click the Submit button to start processing. This will trigger the CRISPR tool to analyze the specified genes.
  3. You will be redirected to a progress page where you can monitor the status of the analysis.
  4. Once the processing is complete, you will be directed to a page displaying the results in a table format.
  5. If you encounter any issues or need help, please refer to the documentation or contact support.

Note: Make sure that the gene IDs you enter are valid and formatted correctly.
